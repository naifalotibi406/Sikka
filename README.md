# 🚆 Sikka — Train Reservation System

> **سِكَّة** — A bilingual train booking platform for Saudi Arabia's railway network.

Sikka is a full-featured train schedule and reservation management system built as a university project for **CS1350 — Software Engineering 1** at Imam Mohammad Ibn Saud Islamic University.

## ✨ Features

- 🔐 Three user roles — Admin, Staff, Customer
- 🌐 Bilingual interface — Arabic (RTL) and English (LTR)
- 🎫 Smart booking — Prevents double-booking
- 💺 Business / Economy + Adult / Child pricing
- 📊 Live dashboard with revenue and occupancy
- 🌙 Saudi heritage dark mode
- 🔒 SHA-256 password hashing

## 🛠️ Tech Stack

- **Backend:** Python 3.10+, Flask
- **Database:** SQLite
- **Frontend:** HTML, CSS, Jinja2
- **Architecture:** MVC + Layered

## 🚀 How to Run

```
pip install -r requirements.txt
python main.py
```

Open: http://127.0.0.1:5000
